# UX_ATM_Moovit
Si presenterà un'accurata analisi di usabilità su due tra le applicazioni più note e contrapposte nel panorama della mobilità milanese: da un lato ATM Milano, l’applicazione ufficiale dell’omonima società che gestisce i mezzi nel capoluogolombardo e dall’altro Moovit, l’applicazione più usata a livello mondiale per i quanto riguarda i trasporti pubblici.
